# Digital Business Card

シンプルなデジタル名刺管理アプリです。
supabaseをバックエンドに使用し、毎日6時にユーザー情報尾９ｋ自動で削除うするバッチ処理を組み込んでおります。

## 機能

- ユーザー登録 / 更新 / 削除
- ユーザーのスキル管理
- 毎朝 6 時に前日のユーザー情報を削除するバッチ処理
- フロントは Vite + React を使用
